Software Creativo

Magical Voxel

En este software podemos comenzar a crear a traves de una geometría, extrayendo o agregando superficies, de esta forma podemos obtener resultados complejos y poligonales para la creación 3D.

Éste software puede ser útil para la creación de objetos para animación, juegos o cómics. Un ejemplo de un juego creado con éste programa es Monument Valley, que aparece en la primera imagen en donde hay que atravesar un laverinto tridimencional.

Monolith

Éste software funciona a base de pixeles tridimencionales, en donde se puede crear a pulso superficies objetos y estructuras. Un voxel es un pixel volumétrico y ésta es la unidad de medida que utiliza.

Por otra parte también se pueden generar fijuras a taves de fórmulas matemáticas y multiplicadores.

xDesign

Explore, Collaborate, Innovate. Esos son las bases de este software en donde puedes trabajar en conjunto utilizando una nuve de información. No utiliza modelación paramétrica si no que ofrece una variedad de herramientas para realizar los modelos 3D. Se pueden dividir los modelos para superficies complejas, sketches o geometrias.

Maya

Crea mundos, personajes complejos y efectos especiales. Sofware de rendering profesional, variedad de superficies y materialidades para tus modelos.
