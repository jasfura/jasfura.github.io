Magical Voxel

En este software podemos crear y editar modelos 3D de 8 bits, siendo una interfaz intuitiva lograremos aprender a utilizarla sin mayor problema. Para resumir un poco esta interfaz dejaré este punteo:

  • Procesador de trazados de ruta incorporado
  • Diferentes tipod se exportación
  • Interfaz open source personalizable
  • Multiples tipos de pincel y tamaños de voxels
  • Formas y patrones prefabricados
  • Se pueden establecer atajos personalizados
  • Renderizado incorporado con texturas y ajustes de camaras
  • 100% gratuito y sin licencia comercial

Demo

1) Paleta de Colores

En esta parte podemos ver los colores disponibles para poner en nuestro modelo, tenemos 3 paletas que vienen incluidas con el software y además podemis descargar y agregarle las paletas que queramos.

2) Herramientas Básicas

En esta sección es donde editamos nuestro modelos, la primera es attach, en donde agregamos voxels para rellenar o modelar nuestro diseño. La segunda es Erase, en donde puedes partir de un cubo e ir eliminando los voxels para modelar. La tercera es paint en donde seleccionamos los colores dentro de la paleta y podemos cambiarle el color a nuestro modelo. La ultima es move, en donde podemos mover el modelo ante un eje.

3) Herramientas de Movimiento

Con estas herramientas podemis mover nuestro modelo mediante un eje, o un espejo. De esta dorma podemos controlar en que parte quedan nuestros detalles.

4) Tipos de Vista

Con estas opciones podemos elegir como veremos nuestro modelo mientras lo estamos editando. Podemos ver la grilla, los límites los cuadros y diferentes tipos de vista para facilitar nuestra edición.

5) Herramientas 1

En esta sección tenemos una variedad de elementos para editar nuestra zona de trabajo.

  • Zero: partir desde cero y poder ir rellenando con la herramienta attach del punto 2.
  • fill: con esto podemos rellenar el cubo al que teníamos inicialmente llenando los espacios que ya habíamos editado para comenzar a borrar voxels con la herramienta erase del punto 2.
  • Full: con esta herramienta también logramos que el cubo quede en su totalidad.
  • Inv: sirve para invertir el modelo, de forma vertical
  • Mirr: Sirve para voltear el modelo de manera horizontal, haciendo un espejo de este.
6) Herramientas 2

Esta segunda sección de herramientas nos permite ritar, voltear y crear una secuencia de nuestro modelo.

7) Herramientas 3

Con etas herramientas de shape podemos modificar nuestromodelo o integrar nuevas figuras listas que vienen con el software como lo son un cilindro, esfera, una pirámide o un labertinto.

8) Herramientas 4

Con estas opciones podemos editar detalles de nuestro modelo.

  • Shell: de la un grosor hueco a nuestro modelo
  • Fract: Fragmenta nuestro modelo
  • Dilat: Engruesa nuestro modelo, entre más veces lo presionemos más grueso quedará.
9) Tipos de cámaras y vistas

Aquí podemos elegir la vista que queremos darle a nuestro modelo, como lo es ortogonal, libre, en perspectiva.

10) Modelos Prediseñados

En esta sección podemos crear a partir de modelos pre diseñados, viene con una variedad de opciones desde modelos 3d y tipo 2d pintados por voxel.

11) Tipos de Exportación

Cuando tengamos finalizado nuestro modelo y queramos exportarlo hacia otro software o alguna máquina de impresión 3d tenemos una variedad de opciones.

  • Obj
  • Ply
  • Md
  • Xraw
  • Slab
  • Qb
  • Iso
  • 2d
  • Bake

Render

En esta segunda ventana podemos realizar cambios y agregar detalles para crear un render. El beneficio de este software es que el sistema de renders viene incluido por lo que no es necesario exportarlo a otra aplicación para realizar uno. Los renders que ofrece se encuentran sobre un plano circular como si fuera un mundito.

1) Iluminación

En esta parte podemos editar el tipo de iluminación que tendrá nuestro modelo, editando la fuerza con la que esta se ve en nuestro render.

2) Luz solar

Podemos editar y elegir como se verá la luz solar que impacta verticalmente nuestro modelo.

3) Diferentes tipos de iluminación

Aquí podemos elegir y determinar la cantidad de luz y sombra que generará, además de diferentes tipos de luz como solar, del cielo y artificial.

4) Botón de captura

Es aquí donde capturamos la imagen cuando tenemos listo nuestro seteo. Es importante esperar a que todo se cargue correctamente antes de presionar esto ya que generará la imagen dependiendo como se previsualiza.

5) Tipos de materiales

Por otra parte nos permite elegir diferentes tipos de materualidades para nuestro modelo.

  • Difuso
  • Metal
  • Vidrio
  • Emisivo
6) Profundidad y material

Podemos seleccionar la funcion que queremos que el render nos entregue, como puede ser difuso o estable.

En este software podemos comenzar a crear a traves de una geometría, extrayendo o agregando superficies, de esta forma podemos obtener resultados complejos y poligonales para la creación 3D.

Éste software puede ser útil para la creación de objetos para animación, juegos o cómics. Un ejemplo de un juego creado con éste programa es Monument Valley, que aparece en la primera imagen en donde hay que atravesar un laverinto tridimencional.

Monolith

Éste software funciona a base de pixeles tridimencionales, en donde se puede crear a pulso superficies objetos y estructuras. Un voxel es un pixel volumétrico y ésta es la unidad de medida que utiliza.

Por otra parte también se pueden generar fijuras a taves de fórmulas matemáticas y multiplicadores.

xDesign

Explore, Collaborate, Innovate. Esos son las bases de este software en donde puedes trabajar en conjunto utilizando una nuve de información. No utiliza modelación paramétrica si no que ofrece una variedad de herramientas para realizar los modelos 3D. Se pueden dividir los modelos para superficies complejas, sketches o geometrias.
